HTML Quick Reference Guide

The list below covers all of the tags we've covered so far.

so far we have learnt..

  1. Basic Tags: the "<p>" tag defines a paragraph. For example:<p> Hello World</p>

  2. Heading Tags: Headings have 6 levels. From h1 the largest to h6 the smallest. For example: <h1>Title</h1> to display the title while <h6>Subheading</h6> for the smallest subheading.

  3. List Tags: There are two types of lists: ordered "<ol>" and unordered <ul>. Take a look at this list for example. This list used <ol>

  4. A href: the "<a>" tags are used to create links.An example for a link tag is this. <a href="lalaland-index.html">HOME</a>.
